// 00 — CORE AREAS

Key Areas of Cybersecurity

Network Security

Protecting infrastructure, firewalls, intrusion detection and securing data in transit.

Endpoint Defence

Securing devices and systems from malware, ransomware and unauthorized access.

Authentication

Identity verification, biometrics, MFA and access control mechanisms.

Encryption

Cryptography, SSL/TLS, hashing and protecting data at rest and in transit.

Penetration Testing

Ethical hacking, vulnerability scanning and simulated attacks to find weaknesses.

Server Security

Hardening servers, patch management and protecting cloud infrastructure.

// ABOUT THIS PROJECT

Who Made This

// The Project

Project Cybersecurity

This site was built as a personal project to collect the best cybersecurity learning resources in one place — no ads, no paywalls, no filler. The goal was simple: create something that actually helps people break into the security field without having to spend hours hunting across the internet for where to start.

Every link, tool, video, and certification on this site was hand-picked based on what's genuinely useful and respected in the industry. Nothing is listed because it paid to be here. If it's on this site, it's because it's actually worth your time.

The design takes heavy inspiration from real security operations center dashboards and terminal interfaces — because if you're going to learn about hacking, the website might as well look the part.

// The Creator

Generated by AI — Completed by Meris

The initial version of this site was generated by Claude — an AI assistant developed by Anthropic. Claude produced the base HTML, CSS, and JavaScript structure from a description of what the site should look like and contain.

From there, Meris took over — reviewing every section, modifying the content, expanding the resource lists, adjusting the design, and completing the final version you see here. The direction, the topic selection, and the finishing work are all Meris's own.

Using AI as a starting point and then building on top of it is a legitimate and increasingly common approach in tech — what matters is understanding what you've built and being able to own it. Meris can explain every section of this site. That's the point.

// Built With

Pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools, no dependencies. Fast, lightweight, and works anywhere. The fonts are from Google Fonts. Everything else is written by hand.

// Disclaimer

Everything on this site is for educational purposes only. Cybersecurity skills are tools — they're only as ethical as the person using them. Only ever practice on systems you own or have explicit written permission to test. Unauthorized access is illegal everywhere.

// Last Updated

Content was reviewed and updated in 2026. The security landscape moves fast — always cross-reference resources, especially advisories and tool versions. What was current last year may be outdated today.
